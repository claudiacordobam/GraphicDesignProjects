# 🎨 Graphic Design Projects
## 📌 Overview
This repository contains a collection of graphic design projects developed as part of my Multimedia studies at UOC. The work explores branding, editorial design and digital layouts with a focus on visual communication and composition.

---

## 🖌️ Projects Included

### 🧘 Yoga App Logo Design
A logo created for a mobile yoga application, focusing on balance, simplicity and brand identity.

### 🔗 Icon Design & Creative Commons Project
An icon developed by combining and adapting open-license resources found online. This project explored the use of Creative Commons licenses and the importance of ethical design practices.

### 📖 Editorial Design
A layout-based project focused on typography, hierarchy and visual composition for editorial content.

### 📰 Newsletter & Web Layout Design
Design and structure of digital layouts for a newsletter and a website, focusing on readability, consistency and user experience.

---

## 🎯 Focus Areas
- Branding and visual identity  
- Typography and composition  
- Digital layout design  
- Ethical use of open-source resources  
- Visual communication
